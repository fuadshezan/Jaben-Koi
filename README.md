# Jaben-Koi
A platform where people can buy their inter-city or local bus tickets. People can search for the available buses according to their destination. If you input the starting point and destination, this website will show the available buses of the route and also the bus stops using the Google Maps API. It calculate the cost according the distance and show the arrival time of the nearest bus stopage.

## Built With

This website built with some language and frameworks. They are:

- HTML 5
- CSS
- Bootstrap
- JavaScript
- Ajax
- PHP
- Google Map API
